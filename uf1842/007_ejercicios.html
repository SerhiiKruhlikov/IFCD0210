Ejercicios Javascript

1. Escribe una función que calcule el área de un rectángulo.
Recibe como parámetros el largo y el ancho.

2. Hacer una función que calcule la tabla de multiplicar de un número que recibe como parámetro. 

			1 X 5 = 5
			2 X 5 = 10

3. Escribe una función que cuente las vocales de una cadena que recibe como parámetro.

4. Escribe una función que reciba como parámetro un array de números y devuelva otro array que contenga solo los positivos.

5. Hacer una función que sume todos los números impares menores a uno dado como parámetro.

6. Hacer una función que encuentre los números comunes a dos listas pasadas como parámetros y los devuelva en otra lista.

7. Hacer una función que cuente los dígitos de un número

8. Hacer una función que devuelva la cadena más larga de las contenidas en un array de cadenas que se pasa como parámetro.
